CSS Animations & Transitions

1. Transition Example

Hover over the box below to see the background color transition smoothly:

transition: background-color 0.5s ease;

2. Animation Example

This box slides in from the left using keyframes and animation:

@keyframes slideIn {
  from { transform: translateX(-100%); }
  to { transform: translateX(0); }
}

.animated-box {
  animation: slideIn 2s ease-in-out infinite alternate;
}

3. Accessibility with prefers-reduced-motion

For users who prefer reduced motion, all transitions and animations are disabled automatically using media queries:

@media (prefers-reduced-motion: reduce) {
  * {
    animation: none !important;
    transition: none !important;
  }
}

Advantages of CSS Animations

  • Performance: Uses GPU acceleration for smoother animations
  • Ease of Use: Simple syntax and no JavaScript required
  • Flexibility: Works for transitions, transforms, fades, rotations, etc.
  • Browser Compatibility: Widely supported with graceful degradation
  • Accessibility: Can enhance user experience with visual feedback
